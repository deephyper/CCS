\doxysubsection{Detailed Description}
Hyperparameters are parameters the when grouped together define a tuning context (see \mbox{\hyperlink{context_8h}{context.\+h}}). 

Most hyperparameters can be sampled according to a distribution\textquotesingle{}s dimension (see \mbox{\hyperlink{distribution_8h}{distribution.\+h}}).
