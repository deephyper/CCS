\doxysubsection{Detailed Description}
An objective space is a context (see \mbox{\hyperlink{context_8h}{context.\+h}}) defining a set of hyperparameters. 

Objective space also define a list of expressions (see \mbox{\hyperlink{expression_8h}{expression.\+h}}) over those hyperparameters called objectives.
