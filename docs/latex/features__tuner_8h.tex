\doxysubsection{Detailed Description}
A CCS features tuner defines an ask and tell interface to optimize an objective space (see \mbox{\hyperlink{objective__space_8h}{objective\+\_\+space.\+h}}) given a configuration spaces (see \mbox{\hyperlink{configuration__space_8h}{configuration\+\_\+space.\+h}}) and a features space (see \mbox{\hyperlink{features__space_8h}{features\+\_\+space.\+h}}). 

The tuner will propose configurations (see \mbox{\hyperlink{configuration_8h}{configuration.\+h}}) and the user will return features evaluations (see \mbox{\hyperlink{features__evaluation_8h}{features\+\_\+evaluation.\+h}}).
