\doxysubsection{Detailed Description}
A CCS tuner defines an ask and tell interface to optimize an objective space (see \mbox{\hyperlink{objective__space_8h}{objective\+\_\+space.\+h}}) given a configuration spaces (see \mbox{\hyperlink{configuration__space_8h}{configuration\+\_\+space.\+h}}). 

The tuner will propose configurations (see \mbox{\hyperlink{configuration_8h}{configuration.\+h}}) and the user will return evaluations (see \mbox{\hyperlink{evaluation_8h}{evaluation.\+h}}).
