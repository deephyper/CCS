Detailed Description

A features space is a context (see context.h) defining a set of hyperparameters.

Validate that a given value at the given index is valid in a features space, and return a sanitized value.

Parameters
[in]features_space
[in]indexthe index of the value in the features_space
[in]valuethe datum to validate
[out]value_reta pointer that will contain the validated value. If value is a string value_ret will contain a non transient string.
Returns
CCS_SUCCESS on success
-CCS_INVALID_OBJECT if features_space is not a valid CCS features space
-CCS_OUT_OF_BOUNDS if index is greater than the number of hyperparameters in features_space
-CCS_OUT_OF_MEMORY if there was a lack of memory while memoizing a string
-CCS_INVALID_VALUE if the value did not validate or if value_ret is NULL
